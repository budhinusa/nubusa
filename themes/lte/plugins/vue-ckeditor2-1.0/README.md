# vue-ckeditor

> Ckeditor directive using Vue.js

![](screenshot.png)

## Notes
- Works with Vue.js 1x. Support for [Ckeditor Vue 2x](https://github.com/dangvanthanh/vue-ckeditor).

## Build

```
npm install
```

Run application

```
npm run dev
```
